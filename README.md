## MEjBNS - Full-Stack JavaScript Using MongoDB, Express.js, jQuery, Backbone.js, Node.js, Socket.io

## Before install

(Here is an example installation for Ubuntu 16.04)

```sh
$ sudo npm install gulp -g
```
## Install

```sh
$ npm install
```

## Start

```sh
$ npm start
```

## Build frontend 

```sh
$ npm run build-front
```
